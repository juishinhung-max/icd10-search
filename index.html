關於本站

本站是為台灣家庭醫學科醫師設計的 ICD-10-CM 代碼速查工具，目的是讓查詢官方中文譯名的流程更快、更直覺。所有代碼皆附中文名稱、英文原文與分類。

使用方式

  • 搜尋：在搜尋框輸入關鍵字，支援中文（糖尿病）、英文（diabetes）、代碼（E11）三種查詢方式，具容錯能力
  • 篩選：點選分類按鈕可限定特定類別
  • 我的最愛：點擊代碼卡片旁的 ★ 圖示，加入個人常用清單
  • 複製：點擊複製圖示可直接將代碼複製到剪貼簿

資料來源

本站代碼對照以衛生福利部中央健康保險署公告的 ICD-10-CM 中文譯本為基礎，目前涵蓋家醫科最常用的代碼，並持續擴充。

  • 健保署 ICD-10-CM 資料站
  • 醫師公會全聯會 ICD-10-CM 查詢系統

免責聲明

本站僅供醫療人員參考使用，代碼對照可能因官方修訂而有差異，請以健保署最新公告為準。本站不取代臨床判斷或專業醫療建議。

貢獻與回饋

如發現錯誤或想建議加入新代碼，歡迎透過 GitHub 回報。
